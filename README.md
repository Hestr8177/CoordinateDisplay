# Coordinate Display Mod

The Coordinate Display mod shows your current coordinates in a fully customizable way. You can configure it easily using Mod Menu, or manually by editing the configuration file: config/hestr8177_coordinate_display_config.json).
Note: This folder and file may not appear until you have launched the game with the mod installed at least once.
You can customize the following settings:

### Settings
- Text Alpha: Transparency of the text
- Text RGB: Color of the text
- Text Shadow: ON/OFF
- Amount of decimals: Number of decimal places for coordinates
- Text Style: Font/style of the text
- Text Spacing: Spacing between letters
- Background Shadow: ON/OFF
- Background Apha: Transparency of the background
- Background RGB: Color of the background
- Background Padding: Size of the background padding
- Position: Location of the coordinate display on screen
- Central Locking: Lock position horizontally and/or vertically
